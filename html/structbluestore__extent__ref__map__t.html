bluestore_extent_ref_map_t::debug_len_cnt bluestore_extent_ref_map_t::debug_peek ( uint64_t  offset) const

Inspects reference region at specified offset. If reference region is located, returns its reference count and distance from offset to the end of the region. If reference region is not located, returns 0 reference count and distance to the next region. If offset is beyond last region, returns 0 for both reference count and the distance.
